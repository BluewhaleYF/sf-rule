# sf-rule
专为 Blue Archive 亚服及国服玩家，Solsynth 产品用户优化的 Shadowrocket 代理分流配置

## 使用指南

 - 打开 Shadowrocket
 - 进入“配置”页面
 - 点击右上角加号
 - 在弹出菜单中粘贴网址
 ```
 https://raw.githubusercontent.com/BluewhaleYF/sf-rule/refs/heads/main/sf-rule.conf
 ```
 - 点击“下载”
 - 下载完毕后点击`sf-rule.comf`
 - 在弹出菜单中选择“使用配置”
 - 进入“首页”页面
 - 在上方“全局路由”设置中选择“配置”
